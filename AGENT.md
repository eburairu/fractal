# AGENTガイド

本書はリポジトリ全体に適用されるエージェント向けのベストプラクティスをまとめています。

## スコープと優先順位
- このAGENT.mdのスコープはリポジトリ全体です。
- 今後`AGENTS.md`または`AGENT.md`が追加された場合、より深いディレクトリにあるものが優先されます。
- システム/デベロッパー/ユーザーからの直接指示は本書より優先されます。

## 作業・コミット方針
- 変更は最小限にとどめ、不要なファイル追加や既存ファイルへの影響を避けます。
- 必要に応じてREADME等へ参照を追記する際は、記載場所と理由を明確にします。

## 返答と思考プロセス
- エージェントの回答および思考プロセスは日本語で出力します。

## コントリビューションメモ
- 可能な限り簡潔かつ自然な日本語で記述・校正してください。
- テストや確認コマンドを実行した場合は、結果を明示的に報告します。
